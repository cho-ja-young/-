# club-matching

## 개발 목적

학우들의 동아리 선택에 도움을 주고 알림제에 참여하지 못한 학우들도 동아리에 대한 정보를 접할 수 있도록 최근 유행하는 테스트 형식의 프로그램을 제공하고자 함. 
간단한 테스트를 통해 학우들은 자신에게 맞는 동아리를 추천받음과 동시에 해당 동아리들에 대한 정보를 접할 수 있음.


## 개발 스택

- html
- css
- javascript


## 개발 일정

- 1/20 (일) 기획안 및 자료 받기
- 2/7 (수) 1차 시안 전달
- 2/9 (금) 피드백
- 2/18 (일) 2차 시안 전달
- 이후 배포, 테스트 진행
- 런칭
